## Vendredi 3 Avril

5x + 3y + 3x + 20 + 10y + 30 = 180

8x + 13y = 130

130 - 13y = 13 (10 - y) doit être divisible par 8. La seule possibilité est y = 2.

On en déduit que 8x = 104 et x = 13.

> réponse: x + y = 13 + 2 = 15
