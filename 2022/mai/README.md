## Mardi 3 Mai

Soient p = nombre de pots et c = nombre de crayons.

Mise en équation:

- 4 × (p - 1) = c
- 3 × p + 1 = c

3 p + 1 = 4 p - 4 ⇒ p = 5 ⇒ c = 16

> réponse: 16 crayons et 5 pots

## Jeudi 5 Mai

En n jours, Jean dépense 1+2+3+...+n = n(n+1)/2 €.

- n (n + 1) / 2 = 210
- n² + n - 420 = 0
- n = 20 (ou n = -21 mais on ne peut pas revenir dans le temps)

> réponse: 20 jours
