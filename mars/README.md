## Mercredi 3 Mars

Il faut que x¹⁰ - 4 soit divisible par x². Donc x² doit diviser 4.

Donc x = ±1 ou x = ±2

Si x = ±1, 1 + k + 4 = 0  ⇒ k = -5
Si x = ±2, 1024 + 4k + 4 = 0  ⇒ k = -1028 / 4 = - 257

> réponse: -5 et -257

## Jeudi 4 Mars

- volume sphère: 4/3 πR³
- surface sphère: 4πR²

V₂ / V₁ = 2 = (4/3 πR₂₁³) / (4/3 πR₁³) = (R₂ / R₁)³ = 2

S₂ / S₁ = (4πR₂²) / (4πR₁²) = (R₁ / R₂)² = 2⁽²/³⁾ = ∛4

> réponse: ∛4

## Mardi 9 Mars

- Jawas = mensonge ⇒ ils répondent 2 oui 1 non
